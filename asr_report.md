# 任务三：开源语音识别（ASR）调研与实现报告
## 一、调研方案对比（3 种）
方案名称	许可协议	语言与方言支持	模型体量与推理速度	流式 / 实时支持	依赖与部署难度	笔记本 / PC 实测感受
OpenAI Whisper	MIT	99 种语言，含普通话	tiny (几十 MB)，CPU 约 1 秒 / 句；large (几 GB)	需二次开发，仅离线	低（仅需 whisper+ffmpeg）	CPU 运行流畅，中文准确率高，操作简单
Vosk	Apache 2.0	中 / 英，支持方言	几十 MB，CPU 推理＜0.5 秒 / 句	原生支持流式 + 离线	极低（仅需 vosk，无需 GPU）	无卡顿，实时识别延迟低，新手零报错
FunASR（阿里）	Apache 2.0	主打中文，支持多方言	轻量版 (几百 MB)，CPU/GPU 均支持	原生支持流式 + 离线	中等（有一键脚本）	中文准确率最高，CPU 稍慢，有官方教程

## 二、选型理由
[部署难度极低，仅需安装 1 个 Python 库，无需配置 GPU，适合笔记本本地运行；
模型仅几十 MB，下载和运行速度快，无硬件压力；
原生支持流式 / 实时识别，可完成 “麦克风实时输入” 实验；
开源免费可商用，对中文支持友好，准确率满足实验需求。]

## 三、实现环境说明
- 操作系统：Windows 11
- Python 版本：3.9.x
- 硬件配置：CPU i5-xxxx，8G 内存，无 GPU
- 依赖库：vosk==0.3.45
